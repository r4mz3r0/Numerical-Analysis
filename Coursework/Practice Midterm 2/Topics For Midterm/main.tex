                $$y_{0} = f(x_0),y_{1} = f(x_1) $$
                $$L_{1,0} = \frac{(x - x_{1})}{(x_0 - x_{1})},
                L_{1,1} = \frac{(x - x_{0})}{(x_1 - x_{0})},$$
                $$P_1(x) = y_{0}(L_{1,0}) + y_{1}(L_{1,1})$$
                    \item[Degree 2]
                    We want to construct an interpolation polynomial of degree two, that is $n = 2$, therefore we need $n + 1$ points.\\

                $$P_2(x) = y_{0}(L_{2,0}) + y_{1}(L_{2,1}) + y_{2}(L_{2,2})$$
                  \item[ Absolute Error] We want to find the absolute error given $f(.45)$, we have previously found $P_1 \text{ and } P_2$. First we evalute the value of $x = .45$ at out interpolant polynomials. 
                  $$P_{1}(.45) , P_{2}(.45)$$
                  Absolute is as follows
                  $$AbsErr1 = |P_{1}(.45) - f(.45)|$$
                  $$AbsErr2 = |P_{2}(.45) - f(.45)|$$
